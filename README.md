# 壹家宠物预约平台

## 项目简介
这是一个宠物预约平台，
用户可以在平台上预约宠物美容服务，
宠物店可以在平台上查看宠物美容服务的预约订单

### 客户端功能
- 提交预约订单，包括不仅限于
  - 宠物信息
  - 预约时间
  - 服务项目
  - 其他要求备注
  - 联系方式
  - 取宠时间

### 后台管理端
- 查看预约单，并确认
- 完成预约单，并通知用户取宠
